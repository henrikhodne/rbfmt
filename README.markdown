# Rbfmt

Rbfmt formats Ruby programs. It's inspired by
[`gofmt`](http://golang.org/cmd/gofmt/), a similar program for Go projects.

## Usage

    $ rbfmt [flags] [path ...]

## Installation

    $ gem install rbfmt
